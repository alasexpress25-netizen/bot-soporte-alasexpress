# Problema: Cuenta y perfil

## Palabras clave
cuenta, perfil, datos, cambiar, modificar, actualizar, email, teléfono, celular, dirección, nombre, eliminar, borrar, desactivar

## Síntomas comunes
- Quiero cambiar mi email
- Necesito actualizar mi teléfono
- Quiero agregar o cambiar mi dirección
- Quiero eliminar mi cuenta

## Solución

### Para cambiar datos de tu cuenta
1. Abrí la app AlasExpress
2. Andá a "Mi perfil" (ícono de persona)
3. Tocá "Editar perfil"
4. Modificá los datos que necesites
5. Tocá "Guardar"

### Para cambiar tu email
El email es tu identificador de cuenta. Para cambiarlo:
1. Andá a Mi perfil → Configuración
2. Tocá "Cambiar email"
3. Ingresá tu contraseña actual
4. Ingresá el nuevo email
5. Verificá el nuevo email desde el link que te enviamos

### Para agregar/cambiar dirección
1. Mi perfil → Mis direcciones
2. Tocá "+" para agregar nueva
3. O tocá una existente para editarla
4. Podés tener varias direcciones guardadas

### Para eliminar tu cuenta
⚠️ Esta acción es permanente y no se puede deshacer.

1. Mi perfil → Configuración
2. Bajá hasta "Eliminar cuenta"
3. Confirmá ingresando tu contraseña
4. Tu cuenta y datos serán eliminados en 30 días

Si querés recuperar tu cuenta antes de los 30 días, contactanos.
