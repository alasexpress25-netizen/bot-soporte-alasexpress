# Problema: No puedo iniciar sesión

## Palabras clave
login, iniciar sesión, entrar, contraseña, credenciales, acceso, usuario, email, correo

## Síntomas comunes
- La app muestra "credenciales inválidas"
- Se queda cargando infinito al intentar entrar
- Dice que el usuario no existe
- No acepta mi contraseña

## Solución

### Paso 1: Verificá tus datos
- Asegurate de escribir correctamente tu email (sin espacios)
- La contraseña es sensible a mayúsculas/minúsculas

### Paso 2: Restablecé tu contraseña
1. En la pantalla de login, tocá "¿Olvidaste tu contraseña?"
2. Ingresá tu email registrado
3. Revisá tu bandeja de entrada (y spam) por el link de recuperación
4. Creá una nueva contraseña

### Paso 3: Verificá tu conexión
- Asegurate de tener conexión a internet estable
- Probá con WiFi si estás usando datos móviles

### Si nada funciona
Tu cuenta podría estar bloqueada o tener un problema especial. Contactanos para revisarlo.
