# Problema: La app no lee los datos / No carga información

## Palabras clave
datos, información, info, lee, leer, carga, cargar, muestra, mostrar, vacío, vacio, nada, no aparece, no sale, perdí, perdi, desapareció, desaparecio

## Síntomas
- La app no muestra mis datos
- Perdí mi información
- No aparece mi historial
- Los datos no cargan
- La pantalla está vacía

## Solución

### Paso 1: Verificá tu conexión
- Asegurate de tener buena conexión a internet (WiFi o datos móviles)
- Probá abrir otra app que use internet para confirmar

### Paso 2: Refrescá la pantalla
- Deslizá hacia abajo para actualizar (pull to refresh)
- O cerrá y volvé a abrir la app

### Paso 3: Cerrá sesión y volvé a entrar
1. Andá a tu perfil
2. Tocá "Cerrar sesión"
3. Volvé a iniciar sesión con tus datos

### Paso 4: Limpiá la caché (Android)
1. Configuración → Aplicaciones → AlasExpress
2. Almacenamiento → Borrar caché
3. Volvé a abrir la app

### Paso 5: Actualizá la app
- Verificá en Play Store / App Store si hay una actualización disponible

## Nota importante
Tus datos están guardados en nuestros servidores, no se pierden. Si después de estos pasos seguís sin ver tu información, contactanos para revisar tu cuenta.
